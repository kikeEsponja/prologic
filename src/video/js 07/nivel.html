Condicionales.

Ya construimos el programa que se ejecutará si el usuario guarda el valor 1 en la variable decision a traves del símbolo "input", falta ahora escribir o construir el programa que se ejecutará si el usuario escribe un número diferente, en cuyo caso, el flujo tomará el camino del "NO"

La opción que podemos darle es que, en lugar de sumar las variables num1 y num2, las reste, y para eso, le asignaremos a la operación el valor 2, así que en la línea "NO", colocamos otro símbolo "selection" y establecemos la condición decision == 2.

Ahora lo que sigue es una repetición de los tres pasos siguientes, el 3 para la multiplicación y el 4 para la división. El paso en final en el que el usuario no tiene más opciones habrá que colocar en la rama "NO" un mensaje con un "output" que le diga que hay un error o que la opción es inválida.
