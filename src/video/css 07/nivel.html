Construyendo una página web desde cero.

Utilizando el wireframe que tenemos, vamos a construir una página web sencilla y estática (estática significa que no tiene interacción con el usuario, para lo que se usa un lenguaje de programación como javaScript).

Comenzaremos creando el espacio de trabajo y los archivos. por razones prácticas, seguiré los mismos pasos del video.

creamos la carpeta musica y dentro de musica nuestro archivo index.html. Abrimos el archivo con el bloc de notas y comenzamos a escribir las etiquetas que ya conocemos:

  • "!DOCTYPE" "html"
  • "head"
    • "meta" "charset=UTF-8"
    • "meta" "name=viewport" "content=width=device-width, initial-scale=1.0"
    • "link" "rel=stylesheet" "href=estilos.css"
  • "header"
    • "nav"
      • "div"
        • "h3" nombre
        • "a" "href=enlace1" link 1
        • "a" "href=enlace2" link2
  • "body"
    • "main"
      • "section"
        • "img" "src=imagenes/"
        • "img" "src=imagenes/"
        • "img" "src=imagenes/"
