Construyendo una página web desde cero.

Encerramos cada "input" dentro de un "div" para alinearlos, luego agregamos la propiedad "color" y el valor "white" al elemento "body" para que todos los textos tomen ese color y eliminamos las reglas para ".link", hacemos lo mismo con el elemento "a" y aumentamos su tamaño con la propiedad font-size; con la propiedad text-decoration y el valor "none", eliminamos el subrayado

también centraremos el elemento "footer" con las reglas "display:flex", "flex-direction:column" y "align-items:center". Al contenedor de las imágenes le agregamos el atributo "class" y le nombramos "galería" luego le aplicamos reglas para manipular los elementos

Una vez centradas las imágenes, agregé a la clase galeria las propiedades flex-wrap: wrap para que las imágenes se posicionen en orden, y un margen de 1% al elemento "img" para que se separen un poco. También agregamos la propiedad border con valores 2px, solid, blue.

    Este será el documento index.html:

    Este será el documento estilos.css:

  • .galeria{display:flex; align-items:center; justify-content:center; flex-wrap: wrap; margin:1%;}
  • .drum{width:340px;}
  • img{width:250px; border-radius: 10px;}
  • body{background:url('./imagenes/banda2.jpg'); color:white;}
  • /* .link{color: #66aa00} */
  • "h3"{position: absolute;} la propiedad position con el valor absolute reduce el espacio que opcupa el elemento a su tamaño real, contrario al valor "relative" que ocupa todo el ancho de la pantalla
  • .barra-navegacion{display: flex; justify-content:flex-end; height: 50px; align-items: center;} la propiedad display con el valor flex nos abre un panel de opciones para posicionar nuestro(s) objeto(s). Una de esas opciones es "justify-content" y se le puede dar uno de cuatro valores básicos: centrado, arriba, abajo, izquierda y derecha.
  • .formulario{display:flex; flex-direction:column; align-items:center; justify-content:center;}
  • a{ text-decoration: none; color:#ffffff; font-size: 20px;}
  • footer{display:flex; flex-direction:column; align-items: center;}
