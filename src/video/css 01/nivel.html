Qué es CSS.

CSS es el acrnimo de Cascade Style Sheet (hoja de estilos en cascada (porque así lo lee la máquina, en cascada)), y sirve para darle a nuestra página estilos personalizados como la posición de los textos, cajas y botnes, dar formas geoméricas a las cajas, manipular tamaños, darles color, animaciones entre otras cosas.


el tema principal de CSS son los selectores, ahora explicaré los más comunes, comenzando por el más común de todos:

Selector de elementos (de etiquetas). Se usa para seleccionar la etiqueta que será afectada, vale decir que afectará a todas las etiquetas con el mismo nombre.

Para construirlo es tan sencillo como escribir el nombre de la o las etiquetas que queremos afecta, por ejemplo "body" luego un par de llaves {} y dentro de las llaves, las reglas que deseamos, estas reglas tienen el siguiente formato: regla: valor;

De igual forma se respetan las reglas del indentado.
