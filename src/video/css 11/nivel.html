Construyendo una página web desde cero.

Continuamos construyendo nuestra página web, haciendo uso de dos documentos diferentes, uno para HTML y otro para CSS.

Podemos usar la consola para ver cómo queda nuestra página en tiempo real, aunque no afecte realmente, es decir, al refrescar la página, los cambios se perderán, pero podremos aplicarlos en nuestro documento.

    Este será el documento index.html:

  • "!DOCTYPE" "html"
  • "head"
    • "meta" "charset=UTF-8"
    • "meta" "name=viewport" "content=width=device-width, initial-scale=1.0"
    • "link" "rel=stylesheet" "href=estilos.css"
    • "nav"
      • "h3" "class=link" nombre
      • "div"
        • "a" "href=enlace1" "class=link" link 1
        • "a" "href=enlace2" "class=link" link 2
        • "a" "href=enlace3" "class=link" link 3
  • "body"
    • "main"
      • "section"
        • "img" "src=imagenes/inst1.jpg"
        • "img" "src=imagenes/inst1.jpg"
        • "img" "src=imagenes/inst1.jpg"
        • "img" "src=imagenes/inst1.jpg"
        • "img" "src=imagenes/groupies.jpg"
      • "section"
        • "form"
          • "input" "type=text"
          • "input" "type=email"
          • "input" "type=number"

    Este será el documento estilos.css:

  • .drum{width:340px;}
  • img{width:250px;}
  • body{background:linear-gradient(red, blue); background-repeat:no-repeat;}
  • .link{color: #66aa00}
  • "h3"{position: absolute;} la propiedad position con el valor absolute reduce el espacio que opcupa el elemento a su tamaño real, contrario al valor "relative" que ocupa todo el ancho de la pantalla
