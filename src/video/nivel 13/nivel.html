TABLE, SELECT y OPTION

Llegamos a este punto, lo que significa que ya pueden crear páginas HTML con bastante autonomía, este último tramo es la antesala al siguiente tema: CSS. pero antes, veremos una etiqueta que tiene cierta relevancia aún, se trata de "table" y sí, sirve para crear tablas.

La forma de crear una tabla no tiene mucha complicación: se crea una etiqueta "table", dentro creamos dos etiquetas "tr" (table row); dentro de la primera creamos dos etiquetas "th" (table head) y dentro de la segunda creamos dos etiquetas "td" (table data):


Las etiquetas "select" y "option" se complementan una a la otra y sirven para crear menús desplegables.

Creamos una etiqueta "select" y dentro creamos tantas etiquetas "option" como opciones necesitemos; cada option tendrá un atributo value con el valor que necesitemos darle para que las opciones tengan contexto, en la imagen dejo un ejemplo básico:

Obviamente esto es una preparación práctica muy básica, pero espero que sirva para que den sus primeros pasos y puedan crear páginas web funcionales, debo acotar que es muy necesario reforzar estos conocimientos con más de una fuente y practicar, de ser posible, a diario.
