Arreglos parte 2.

Construyamos un programa para dar un ejemplo de arreglos, abrimos Raptor y creamos un proyecto nuevo. Declaramos una variable con "assignment" y le llamamos instrumento, lo establecemos con el tipo numérico y luego, usamos otra variable para nuestro arreglo; le nombramos "tren" y le colocamos un par de corchetes, dentro de estos se determina el número de posiciones que tendrá, para esto, usaremos un "input" para que el usuario establezca este dato.

una tercera variable llamada "bucle" se encargará de recorrer el arreglo y la inicializamos con el valor 1. El siguiente paso es colocar un "loop" y asignarle como condición (dntro del rombo) que mientras instrumento sea menor que bucle, se siga repitiendo. Con otro "input", el usuario cargará manualmente los instrumentos; este irá en la rama "no". La sintaxis (forma de escribir) será "tren[bucle] (porque "bucle" es el índice del arreglo)"

Con un nuevo "assignment" creamos el incremento colocando en el primer campo "bucle" y debajo "bucle + 1"
