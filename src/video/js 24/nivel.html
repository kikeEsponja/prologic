OBJETOS.

Los objetos son otras estructuras para almacenar datos de diferentes tipos.

se declaran como una variable normal y luego de la igualdad van dos llaves dentro de las cuales escribiremos las propiedades y valores de nuestro objeto en forma de propiedad: "valor" y separados entre si por comas.

let objeto = {parte1: "tope", parte2:"medio", parte3: "piso"}

para acceder a los componentes o partes del objeto se puede usar la notación de punto: objeto.propiedad

existe la posibilidad de modificar las propiedades del objeto

var persona = {nombre: "juan", edad: 30, ubicacion: "londres"}

luego

persona.edad = 40;

tenemos los métodos ".keys", ".values" y ".entries" que vamos a usar con la palabra reservada "Object" (con la "O" mayúscula)

const atributos = Object.keys(persona);

const valorDelAtributo = Object.values(persona);

const todosLosDatos = Object.entries(persona)


MAP

para construir esta estructura contaremos con la palabra reservada new, de esta forma: let variable = new Map();

las propiedades y valores para este tipo de sistema se agreganluego con el método "set".

variable.set("nombre", "alfonso");

variable.set("edad", 25)
