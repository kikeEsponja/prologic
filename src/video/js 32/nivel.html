Esto ya lo hicieron así que no debería haber problemas, sólo van a agregar las líneas para capturar os elementos HTML mediante su "id" y listo.

el hecho de que en este pequeño y básico cursillo esté usando el método document.getElementById() para capturar elementos, no significa que sea la única forma.

se puede capturar un elemento por su selector CSS, con document.querySelector("#miId") este capturará el primer elemento que coincida

document.getElementsByClassName(".miClase") captura todos los elementos que tengan la class="miClase"

document.getElementsByTagName("div") captura todos los elementos cuya etiqueta sea "div"

document.getElementsByName("name") captura todos los elementos que tengan el atributo name
