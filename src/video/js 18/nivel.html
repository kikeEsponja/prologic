FUNCIONES.

Son bloques de código que podremos reutilizar cada vez que los necesitemos. Esto, claramente nos permite optimizar nuestros archivos evitando repetir comandos y operaciones. Para construirlas se escribe la palabra reservada "function" seguida del nombre que queramos darle, le sigue un par de paréntesis que pueden estar vacíos o contener algún parámetro y finalmente un par de llaves dentro de las que irá nuestro programa.

function nombre(){ programa }

Luego, para invocar a la función, solo se escribe su nombre seguido de los paréntesis: nombre();

En la consola pueden probar el ejemplo:

function mensaje(){alert("esto es una función");}

importante el punto y coma al final


trabajar con parámetros también es muy útil, utilizamos la palabra reservada "return" para que la función retorne algún resultado.

function suma(a, b){return a + b}

también pueden probarlo en la consola, reemplacen las letras del paréntesis por números cualquiera. Si lo dejan vacío, retornará NaN (Not a Number) del resto, puede ser cualquier tipo de dato. Hay que acotar que las variables declaradas dentro de una función sólo servirán dentro de la función, tratar de acceder a ellas desde fuera, arrojará un error.
