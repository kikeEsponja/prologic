DOM

Es el acrónimo de Document Object Model y es la representación programática que hace el navegador de la estructura de una página web, y esa estructura ya sabemos construirla con HTML, y javaScript tiene el poder de interactuar con ese DOM

Bien, ahora tenemos dos partes: javaScript y el DOM. Vamos a juntarlas para que puedan trabajar.

exactamente como hicimos en el tema CSS vamos a hacer acá, justo antes de la etiqueta de cierre de "body" vamos a escribir una etiqueta "script" y dentro de esta, escribiremos todo nuestro código de javascript

<script>//tu código de javascript acá</script>

por supuesto que si nuestro código javaScript se hace extnso y no parece sensato ni estético colocarlo dentro de una etiqueta "script" podemos crear un documento aparte como hicimos con el CSS y enlazarlo. Para esto, creamos el documento con extensión ".js" y usaremos de nuevo la etiqueta "script" pero en lugar de encerra nuestro código dentro, le vamos a agregar un atributo "src" y como valor le asignamos la ruta hacia ese documento:

<script src="./ruta_al_documento.js"></script>
