Formularios parte 2

Bien, continuaremos con el tema de formularios. En el tema anterior hablamos de la etiqueta "label". Esta etiqueta tiene un atributo "for" y el valor de este será el mismo valor que tenga el atributo "id" de la etiqueta "input" relacionada.

Como expliqué anteriormente, el atributo "type" y el valor "number" para restringir la entrada de datos a sólo números, también existe el valor "email" para que el dato que se ingrese sea un correo electrónico con un formato válido, el valor "checkbox" crea un cuadro de selección y el valor "radio" crea botones para selección múltiple.

Este último, el valor "radio" requiere los atributos "name" y "value" para la selección múltiple. Para mostrarlo, colocaré dos "input" con atributos "radio", "name" y "value"; el primer "input" tendrá como valor del atributo "name" la palabra "seleccion" y para "value" será la palabra "selec"; para el segundo "input", el valor de "name" será la palabra "seleccion2" y el de "value" será la palabra "selec2"

Y para lograr que con una selección se desmarque la otra, sólo usamos el mismo valor para los atributos "name" y listo.
