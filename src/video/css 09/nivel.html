Construyendo una página web desde cero.

Existe otra forma de aplicar estilos CSS a nuestro documento HTML y es con la etiqueta "style", se coloca dentro de la etiqueta "head". Esto generalmente se usa cuando las reglas son pocas. También agregaremos una etiqueta "label" antes de cada "input" para describir mejor esta última.

Por último, agregamos una etiqueta "aside" que será un contenedor lateral (que podría ser también un "div" pero si vemos la palabra "aside", de forma rápida podremos ubicarla sobre el resto de los div) y dentro anidamos una etiqueta "p" y una "img" con un atributo "alt" que tendrá como valor una descripción de la imagen

  • "!DOCTYPE" "html"
  • "head"
    • "meta" "charset=UTF-8"
    • "meta" "name=viewport" "content=width=device-width, initial-scale=1.0"
    • "link" "rel=stylesheet" "href=estilos.css"
    • "style"
      • img{ width: 200px; }
      • body{ background: linear-gradient(red, blue); }
