Aplicando CSS.

Ya sabemos lo que son selectores, reglas, valores y cómo se usan, ahora vamos a aplicar eso a nuestro documento HTML


Vamos a crear otro documento (igual que hicimos para crear index.html) y lo vamos a guardar con un nombre y la extensión css (nombre.css). Ahora abrimos nuestro archivo index.html y dentro de la etiqueta "head" vamos a agregar la etiqueta "link" que tendrá un atributo "rel" con valor "stylesheet" y un atributo "href" cuyo valor será la ruta a nuestro archivo con extensión css que acabamos de crear.
