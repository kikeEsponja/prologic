Creando un carrusel

Para finalizar con este tema, vamos a crear un carrusel de imágenes (básico, claro) luego ustedes pueden desarrollarlo a fondo y crear cosas más profesionales y complejas. Primero creamos un contenedor para nuestro carrusel (un "div") y le agregamos el atributo class y le damos el valor "carrusel-contnedor", luego dentro, vamos a anidar otro "div" y le agregamos la clase (es decir, atributo class) carrusel-inner. Dentro de carrusel-inner anidamos tres "div" más, cada uno con la clase carrusel-item, dentro cada uno de estos tres "div" vamos a anidar una etiqueta "img" en donde irán tres imágenes, yo usaré las de los instrumentos musicales.

listo, ahora, en un documento aparte (al que también llamaré estilos.css pero ustedes pueden nombrar como quieran mientras luego lo enlacen correctamente) colocaré las reglas para los elementos, a estas alturas deboo asumir que lo entienden del esquema descrito debajo.

    Este será el documento index.html:

  • "!DOCTYPE" "html"
  • "head"
    • "meta" "charset=UTF-8"
    • "meta" "name=viewport" "content=width=device-width, initial-scale=1.0"
    • "link" "rel=stylesheet" "href=estilos.css"
    • "style"
    • "body"
      • "div" "class=carrusel-contenedor"
        • "div" "class=carrusel-inner"
          • "div" "carrusel-item"
            • "img" "src=./imagenes/ins1.jpg"
          • "div" "carrusel-item"
            • "img" "src=./imagenes/ins2.jpg"
          • "div" "carrusel-item"
            • "img" "src=./imagenes/ins3.jpg"

      Este será el documento estilos.css:

    • div{border: 2px solid black; width:50%; background:grey;}
    • h1{font-family: 'algerian'; height: 152px; color:green;}
    • body{font-family: Arial, sans-serif; margin: 0; padding: 0; box-sizing: border-box}
    • .carrusel-contenedor{position: relative; width: 100%; max-width: 600px; margin: auto; overflow: hidden} (esta propiedad oculta las barras de desplazamiento)
    • .carrusel-inner{display:flex; animation: carousel-animation 9s infinite linear;} (establecimos la duración y dirección del carrusel)
    • .carrusel-item{min-width: 100%; box-sizing: border-box:}
    • .carrusel-item img{width:100%; height: auto; display: block;}
    • @keyframes carousel-animation{0%, 100%{transform: translateX(0);}25%{transform: translateX(0);}33.33%, 58.33%{transform: translateX(-100%);}66.66%, 91.66%{transform: translateX(-200%);}}
