EVENTOS.

en cualquier programa existen cosas que pasan, como un clic con el ratón o el presionar de una tecla, que la página se cargue o determinado tiempo transcurra, todo eso (y otros que no he nombrado) se llaman eventos, y podemos manejarlos con javaScript

vamos a ver esto con ejemplos:

  • En el mismo documento creamos una etiqueta "div" para encerrar nuestro ejercicio, esto nos dará una visión objetiva de nuestro trabajo. Creamos dentro de "div" dos "input" cada uno con un atributo "id" y como valor "num1" y "num2" respectivamente.
  • creamos una etiqueta "button" con el atributo "id" y el valor "boton"; como contenido podemos escribir resolver o acción o cualquier cosa.
  • una etiqueta "p" con contenido "resultado"
  • una etiqueta "span" sin contenido, con un atributo "id" de valor "resultado"

ahora en el documento "ejercicio.js" haremos lo siguiente:

  • declarar una variable, podemos llamarle boton, para saber a qué nos estamos refiriendo y capturar el id respectivo, que sería "boton" (en el documento index.html debe estar el id="boton" en la etiqueta "button")
  • declarar la variable para la etiqueta "span" y capturar también su "id"
  • a la variable botón le vamos a aplicar el método .addEventListener() (literalmente traduce agregaEscuchadorDeEventos)
  • esta parte es interesante: dentro de los paréntesis irá entre comillas el evento, en este caso "click" y separado por una coma crearemos una función; esa función creará dos variables: num1 y num2 que guardaran los elementos "input" con los respectivos "id" que están en el documento HTML y les aplicará el método ".value" que recogerá lo que sea que l usuario escriba dentro del "input"
  • como nota adicional, en javaScript también podemos (y luego verán lo necesario que es) comentar código: para una línea se usa un doble slash: // y para varias líneas se usa igual que en CSS /* y dentro escribimos el lo que sea y no afectará nuestro programa */
