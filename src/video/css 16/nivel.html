Más propiedades CSS

Usaremos otra página web en blanco para explicar otras propiedades importantes. Crearemos un documento html y dentro de "body" agregaremos una etiqueta "div" y dentro de esta, agregamos un "h1" con cualquier texto. Dentro de la etiqueta "head" agregamos la etiqueta "style" para ver las propiedades.

dentro de "style" agregamos la propiedad margin, que tiene cuatro variantes:

  • margin-top (arriba)
  • margin-bottom (abajo)
  • margin-left (izquierda)
  • margin-right (derecha)

también podemos escribirlo como margin: xx xx xx xx donde cada par de equis equivale a una posición: el primer par es para top, el segundo es para right, el tercero es para bottom y el último para left

Otra propiedad importante es padding, que también tiene cuatro variantes:

  • padding-top (arriba)
  • padding-bottom (abajo)
  • padding-left (izquierda)
  • padding-right (derecha)

también podemos escribirlo como padding: xx xx xx xx donde cada par de equis equivale a una posición: el primer par es para top, el segundo es para right, el tercero es para bottom y el último para left

    Este será el documento index.html:

  • "!DOCTYPE" "html"
  • "head"
    • "meta" "charset=UTF-8"
    • "meta" "name=viewport" "content=width=device-width, initial-scale=1.0"
    • "link" "rel=stylesheet" "href=estilos.css"
    • "style"
    • "body"
      • "div"
        • "h1" objeto

      Este será el documento estilos.css:

    • *{ margin:0; padding:0; box-sizing:border-box }
    • body{ color:#fff; display:flex; justify-content:center; }
