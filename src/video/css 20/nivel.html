Media queries.

Se construyen colocando el símbolo de arroba seguido de la palabra "media" así: @media y entre paréntesis los diferentes tamaños de pantalla o rangos de tamaño que se van a manejar, enseguida les dejo un ejemplo:

    Este es el documento HTML

  • "!DOCTYPE" "html"
  • "head"
    • "meta" "charset=UTF-8"
    • "meta" "name=viewport" "content=width=device-width, initial-scale=1.0"
    • "link" "rel=stylesheet" "href=estilos.css"
    • "style"
  • "body"
    • "h1" TEXTO

    Este es el documento CSS

  • @media screen and (max-width: 700px){h1{display:none}}
  • Esto quiere decir que cuando la pantalla sea menor a 700 píxeles, la etiqueta "h1" pasará a tener la propiedad display: none, es decir, va a desaparecer

En resumen, los mediaQueries se usan exactamente igual que las reglas que ya conocemos, pero encerradas dentro de un bloque de código que funcionará sólo para el tamaño o rango de tamaños de pantalla que especifiquemos.
