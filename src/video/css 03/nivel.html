Cambiando colores y conociendo otros selectores.

vamos a cambiar de nuevo el color al título y agregar un fondo a toda la página, gris, por ejemplo (recuerden que todo esto se maneja en inglés).

para esto, seleccionamos la etiqueta "body" y establecemos dentro de las llaves, la regla background : grey;

En CSS también podemos comentar código para que podamos escribir sin afectar nuestro proyecto, en este caso, encerramos nuestro comentario entre los símbolos asterisco y barra invertida, así: /* aquí el comentario */

Vamos a crear dos etiquetas "h2" y le vamos a escribir un texto, yo escribiré "trabajando con clases", y al aplicar la regla color: red; ambos "h2" van a cambiar, para cambiar sólo el "h2" que deseamos, usaremos el atributo "class" o "id" con un valor que se nos haga fácil identificar. La diferencia entre estos dos atributos radica en que el atributo "id" no puede repetirse mientras que el atributo class, sí.

por ahora usarmemos el atributo "class" y se lo colocaremos a uno de los dos "h2", luego, en el documento estilo.css escribiremos la regla color: red; sobre el selector de clases "." (es decir, un punto)


el atributo "id" funciona igual, pero para usarlo en CSS se usa el símbolo numeral o hashtag (#) entonces seguido del nombre del id que le hemos colocado al elemento.
