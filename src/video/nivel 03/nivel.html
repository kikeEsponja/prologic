BIEN HECHO!

Es momento de aplicar lo que hemos aprendido y construir nuestra primera página web. Es tan, pero tan sencillo como crear un archivo dentro de nuestra carpeta con el bloc de notas, ponerle un nombre y la extensión html. (sí, así de simple). Yo crearé uno y lo llamaré pagina.

Al hacer doble clic en nuestro nuevo archivo, y según lo que hemos aprendido, la aplicación predeterminada para abrir archivos con esta extensión es el navegador o browser (en inglés). Obviamente lo que veremos será una página en blanco.


Comprendido esto, vamos a poner cosas dentro de este espacio en blanco, entonces es cuando toca un nuevo objetivo: ETIQUETAS

Las etiquetas son el componente principal que conforma el contenido de una página web, vamos a ver dos tipos: simples y compuestas

  • las simples se construyen encerrando el nombre de la etiqueta entre dos símbolos de mayor qué y menor qué. Así: < nombre > <nombre>
  • las compuestas se construyen con una etiqueta simple que abre y otra que cierra; a la etiqueta de cierre, le agregamos una barra invertida o slash antes del nombre. Así: < nombre > </ nombre >
  • Generalmente, este tipo de etiquetas encierra algún tipo de contenido: < nombre > contenido </ nombre >

Teniendo esto claro, podemos hablar del siguiente tema: el INDENTADO

No es más que una sangría, un margen que dejamos para que nuestro código sea entendible, como todo, tiene sus reglas, y para entenderlas, es necesario ver a las etiquetas como unas cajas que contienen cosas, incluso otras etiquetas, veamos cómo funciona el indentado con otro concepto: Padres e hijos (o hijas)

Con el indentado podemos saber cuales elementos son padres y cuales los hijos. Podemos tomar el ejemplo de la etiqueta nombre que creamos anteriormente y la escribimos así: ese espacio que diferencia contenido de las etiquetas "nombre" se llama indentado, y podemos decir que "contenido es hijo (o hija) de "nombre"

ANIDACIÓN es otro término muy usado y sólo quiere explicar la colocación de elementos dentro de otro elemento, así como una caja dentro de otra caja. Crearé una etiqueta llamada "caja" y dentro, anidaré la etiqueta "nombre" Así "caja" es padre de "nombre"
